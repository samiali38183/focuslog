## Features

| Feature | Description |
|---|---|
| Pomodoro Timer | Customizable focus and break intervals with animated SVG ring |
| Session Logging | Automatically logs every completed focus session with timestamps |
| Weekly Dashboard | Visual overview of your productivity across the week |
| Settings | Adjust focus duration, break length, and daily session goals |
| Dark Theme | Clean, minimal dark interface with smooth animations |
| Persistent Storage | Sessions saved to local storage so your data is always there |

## Getting Started

**Use the live app:** Visit [focuslog-ochre.vercel.app](https://focuslog-ochre.vercel.app) — no installation needed.

**Run locally:**

```bash
git clone https://github.com/samiali38183/focuslog.git
cd focuslog
open index.html
```

No dependencies. No build step. Just open `index.html` in your browser.
